# Mylange Programming Language

<img src="https://myriware.space/favicon?line=_770000" alt="Alt text" width="25%" height="auto">

The Mylange Programming Language is the Myriware in-house solution that
allows for all employees to understand and write basic functional code.
Mylange is a hybrid of many other languages, taking the best features
from all and emplying them into one simple language.

Mylange is an object-oriented and functional programming language.
It has strict typing, such as Java or C#.

```mylange
str String => "Hello World!";
```
